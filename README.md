# anaplan-orm
A lightweight Python 3 library that abstracts the Anaplan API into an Object-Relational Mapper (ORM).

## Current Status
🚧 **Under Active Development** 🚧
Currently building the core data transformation and parsing engine. 

## Features
* Extracts XML string payloads directly into Pydantic V2 models.
* Serializes Pydantic models into Anaplan-ready CSV payloads.
* (Coming Soon) Handles Anaplan API authentication and bulk import actions natively.
